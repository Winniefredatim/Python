Variables and simple data types

Adding a variable makes a little more work for the python interpreter, when it processes the first line, it associates the text "Hello World with the varaible message. When it reaches the second line, it prints the value associated with the messageto the screen.

Data types

Data types include:
  • Strings: A string is simply a series of characters inside quotes (the quotes can be single or double).
  • Numbers: This is also another data type which is divided into Integers and Floats.
  • Comments: These are data types in python which allows a programmer to write notes in English within a program and it is indicated using the hash mark (#)

Lists

Lists allows a programmer to store sets of information in one place whether you just have few items or millions of items. Lists are one of Python's most powerful features readily available to new programmers and they tie together many important concepts in programming. A list is a collection of items in a particular order and it is indicated by Square brackets ([]) for example bicycles = [trek, specialized, redline] print(bicycles), if python is to print, it returns [trek, specialized, redline].

If statements

Python's If statement allows you to examine the current state of a program and respond appropriately to the state. At the heart of every if statement is an expression that can be evaluated as True or False and this is called a Conditional test.

Dictionaries

A dictionary is a collection of Key-value pairs. It allows a programmer to connect pieces of related information. It stores limitless amount of information. Each key is connected to a value and you can use a key to access the value associated with that key.

Functions

Functions are named blocks of code desighned to do one specific job. When you want to perform a particular task that you've defined in a function, you call the name of the function responsible for it and if you need to perform that task multiple times throughout the program, you just call the function dedicated to handling that task.
